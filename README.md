# HackclubKeypad
Addition to 60% keyboard including F row and arrow keys

Features
* F row
* Arrow Keys
* Rotary Switch
* LED backlights
* Hotswap
* Bottom Mounted


Materials Needed
* Solder Iron
* 3.5mm standoffs
* EVA foam
