# HackclubKeypad
Addition to 60% keyboard including F row and arrow keys

Features
* F row
* Arrow Keys
* Rotary Switch
* LED backlights
* Hotswap
* Bottom Mounted


Materials Needed
* Solder Iron
* 3.5mm standoffs
* EVA foam

Layout: https://www.keyboard-layout-editor.com/#/gists/ed5083c6de0b27ba4e196096e7faea45
